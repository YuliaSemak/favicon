Node.js Core Server (Development Version)
Це легка реалізація вебсервера на базі стандартної бібліотеки Node.js. Проєкт створений для демонстрації базових принципів обробки HTTP-запитів та роботи з файловою системою без використання зовнішніх фреймворків (як-от Express).
